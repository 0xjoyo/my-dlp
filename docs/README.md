# Screenshots go here.

This folder is intentionally empty in the source tree. The README links to
PNG files in this directory; they are added as the visual design evolves
and uploaded alongside release binaries.

If you'd like to contribute a screenshot:

1. Take a screenshot of the running app (Windows / Linux / macOS).
2. Save it as `screenshot-downloader.png`, `screenshot-spotify.png`, or
   `screenshot-lyrics.png` in this folder.
3. Open a PR.
